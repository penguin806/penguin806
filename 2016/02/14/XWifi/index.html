雪峰Wi-Fi [Shizuku Edition]

发表于 2016-02-14   |  

雪峰Wi-Fi, 免费上网利器, 创建软AP, 只需一键…

软件特点

  1. 操作简单
  2. 体积小巧, 单文件, 不到2M, 方便携带
  3. 资源占用低
  4. 免费, 纯净无捆绑
  5. 兼容Win7及以上系统(不含Vista)
  6. 支持中文SSID
  7. Moe :)

一些细节

使用方法

  • 填入SSID与Key
  • 单击”设置”按钮

  • 在WAN 下拉框处选择你希望「雪峰Wi-Fi」共享上网的网卡
  • 单击”启动”按钮, 等待1~10秒后即可体验免费WiFi之旅!

Q&A

  • Q: 如何显示/隐藏任务栏托盘图标?
    A: 单击程序菜单”选项”->”托盘图标”
  • Q: 程序的各项设置在退出后无法保存, 怎么办?
    A: 「雪峰Wi-Fi」通过注册表来保存各项参数, 在程序退出时会将设置写入注册表, 以便启动时载入配置. 如果无法保存, 请检查是否有安全软件拦截了「雪峰Wi-Fi」的注册表操作
  • Q: 点击「雪峰Wi-Fi」分享按钮后居然卡死了!
    A: 别着急, 多等等. 由于「雪峰Wi-Fi」此处的设计是单线程, 因此程序要在完成分享工作后才能回到消息循环, 这个过程根据您机器的配置大概需要1~10秒
  • Q: 为什么「雪峰Wi-Fi」每次启动都有个这样的框框, 好烦.
    A: 这是微软在Vista之后推出的一项”贴心”设置, 在开启UAC的情况下, 若当前用户非内置管理员Administrator, 而程序又需要权限, 则会弹出一个UAC对话框请求批准, 单击”Yes”就好. 关闭UAC: “Control Panel” -> “Security and Maintenance” -> 左侧的”Change User Account Control Settings” -> 调整为Level 0(Nerver Notify Me)
  • Q: 不喜欢这个软件, 我想彻底清除TA.
    A: 如果你和我一样对自己的电脑有洁癖, 请顺便删除注册表键值HKEY_CURRENT_USER \ SOFTWARE \ X-Wifi

关于背景

软件背景中的小朋友由 Kirino Kasumu 创作, 我在用「CrystalDiskMark」(一款十分好用的硬盘跑分软件 Download Portable Edition) 时偶然发现的, 很喜欢.

鉴于「雪峰Wi-Fi」是一个非商业共享软件, 以及CrystalDiskMark也开放了源代码, 我就这样把图偷拿过来了, 十分感谢CrystalDiskMark的作者Kirino Kasumu. (不知道他会不会怪我_(:3 」∠)_)
